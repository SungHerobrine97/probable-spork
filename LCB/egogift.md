# 人格、E.G.O、饰品

{% hint style="tip" %}

## 本内容可能掺杂其它游戏的内容

如果感觉不适，请勿继续阅读。

{% endhint %}

这里的人格指的是某个角色在其他世界的可能性（或者是另一个人生），正如希斯克利夫过着不同的生活、与不同的人相遇、过着没有凯瑟琳的人生那样。

自身的可能性非常多，但是真正能实现自己人生价值的却只有你自己。

“*如果接受并背负了自己的罪，心灵原本的形状就将显现为E.G.O。*”

## 饰品

这里的饰品可以是某个角色的信物，也可以是一个帮派的道具和其他装备（并非所有物品都是饰品）。

一个饰品的示例如下所示：

```Markdown
## 破损的猫眼石胸针

[第一顺位专属]

自身三技能触发呼吸法暴击时，本次攻击不消耗呼吸法（每3回合可触发一次）。

[不可重复触发] 当鸿园的君主鸿璐在场时，该效果无法触发。
```
