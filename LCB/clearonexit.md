# 退出场景时重置自身状态效果

## 问题

因为部分角色的状态存储在RAM内存中，有的时候因为程序自身设计的问题不得不进行说明。

在Limbus Company原版作品中，人格在结束战斗时会将自身变回初始形态**并清除自身的状态效果**，因此你不得不在战斗被动写一些设定，比如（以蜘蛛巢之刃良秀为例）：

```
战斗中首次登场时，使自身获得0层纠缠与0层断缘（每场战斗都会初始化状态效果）
```

## 对于同人剧情

视觉小说中，你不需要考虑这种设定（~~除非小说作者明确要求~~）。

## 其他游戏

首先这个设定不特定于某些二游，有些角色自身的状态是保存到存档当中（而不是RAM）。
