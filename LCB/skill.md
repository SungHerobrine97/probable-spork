## 强化技能释放条件设定要求

你可以选择积攒资源释放、也可以通过特定的状态效果触发特定的形态，也可以用“如果……就……”这一语句来定义强化技能触发条件（**月计的状态效果甚至可以描述剧情状态**，例如环指浮士德）。

```
状态效果：拘束解除-创作沉浸
1. 替换一组新的基础技能
2. 最小与最大速度值+2
3. 通过基础攻击技能对速度值低于自身的敌方单位造成的伤害+10%
```

此外，还有一些时间点如果放置的恰当好处，还能更好的刻画一个角色，以下仅为常见的技能触发时间节点。

* **拼点胜利**：效果与目标拼点胜利时触发
* **拼点失败**：效果与目标拼点失败时触发
* **使用时**：包含技能获得强化的使用条件，往往有“目标的xxx强度每有x级，使本技能最终威力+y”之类的
* **命中时**：目标闪避失败、未使用闪避、被作为攻击目标且被命中时才会触发的效果，**广域乱射是个例外，因为一次攻击只能命中一个目标**。
* **广域乱射**：首次实装为“R公司-以实玛利”，一个硬币命中一个目标，_至于命中哪个谁都不知道了_。
* **改变本场战斗的战斗音乐**：仅限人格高光或者剧情需要

有些强化技能的条件要求可以根据人格的设计目的来决定：

* 如果具有特定状态，则释放强化技能（例如狂猎希斯克利夫）。

```
如果自身具有杜拉罕，则使用“悲叹，哀恸，破灭吧”
```

* 自身（或者是除自身之外的其他目标）某些状态效果达到特定层数时并且有特定友方单位参战（_或阵亡_），则释放强化技能（例如拉曼却领总督-堂吉诃德。

```
回合开始时，若自身的硬血层数不低于15层且以下单位中有单位参战或阵亡，强化自身最左侧的行动槽中的相应技能。
- 拉·曼却领神父格里高尔：强化1技能
- 拉·曼却领理发师奥提斯：强化2技能
- 自身：强化3技能
- 拉·曼却领公主罗佳：强化守备技能
```

* 特定条件满足时可释放（参考剑契组-默尔索和污血鸿璐）。

```
拼点失败且受到攻击后，释放技能“断汝筋骨”。
```

```
受到使自身体力降至0点的伤害时，本回合自身体力不会低于1点，解除自身因受到伤害而陷入的混乱，本回合自身不会陷入混乱或阵亡，在该回合最后使用技能“污血泣泪-终”。该技能结束后解除葬花楔
```

## 差异

* 在鸣潮中，所有技能均**遵循单方面攻击**的逻辑，而边狱巴士当中，一些技能因演出或者是机制需要往往会设定一些非常逆天的数值或者机制（视觉小说这一块）。因此，脚本就成了剧情演出不可缺少的一环。（_**为什么我和达妮娅玩接好饭的时候到最后我却无法操作了？**_）

> 这里的脚本并不是我们眼中的自动化脚本，有关于特定游戏引擎的脚本请移步到[虚幻引擎5文档](https://dev.epicgames.com/documentation/zh-cn/unreal-engine/overview-of-blueprints-visual-scripting-in-unreal-engine)或[Unity文档](https://docs.unity3d.org.cn/Manual/CreatingAndUsingScripts.html)。有关剧情演出的脚本，可参考一个[Github仓库](https://github.com/zhongcaojieji/TimelineScript)。
>
> 脚本已超出本文所讨论的范围。

## 后记以及反思

截至到目前，我们可能明白为什么一些游戏它的版本周期只有42天（或者更短），以及为什么绝大多数二游都有类似原神的圣遗物、命座、技能强化机制，而月计的边狱巴士却只有人格等级和同步阶段、异想解析阶段这三种。

有时候并不是因为边狱巴士剧情评价好才出现这个文章，而是有的时候同质化二游太多（自原神之后）导致的审美疲劳也导致了写不好边狱巴士同人人格技能。

有时候你可以在必应搜索引擎搜一下“二游剧情制作工业化”来了解一下为什么二游剧情制作现状是什么样子了。

你也可以看一下[这个文章](https://www.chinawriter.com.cn/n1/2025/0312/c419388-40436928.html)来了解为什么。

> 事已至此，看一下良秀对于环指野兽派的评价吧。
