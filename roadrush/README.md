# 暴力摩托同人

本文档仅为部分其他二游非官方同人扩展的补充，此部分不应作为任何一个独立扩展。

## 比赛规则

> 本条目的数据取自[百度百科](https://baike.baidu.com/item/%E6%9A%B4%E5%8A%9B%E6%91%A9%E6%89%98%E8%BD%A6%E6%89%8B/8466535)，有修改。  
> 本规则为《暴力摩托》原版通用规则（第二条除外）。

1. 开始比赛前，你需要选择一个车手出战这场摩托比赛。
2. 比赛所用的摩托由各地区的主管部门提供。  
    **如果比赛所在的地区不提供摩托或当地主管部门无法提供摩托，本场比赛的摩托将由其他地区的部门提供（无需承担赛事经费）。**
3. 所有比赛都提供有战术道具和后勤支援，**车手无需担心自身车辆状况及修车经费**。
4. 每场比赛要求选手在比赛中获得前三名才能晋级更高难度的比赛（特色化比赛除外）。
5. 可以使用战术装备攻击其他车手以获得优势。
